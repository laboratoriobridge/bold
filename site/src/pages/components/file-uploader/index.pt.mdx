# File Uploader

Permite ao usuário transferir e submeter conteúdo do próprio computador. A seleção do arquivo pode ser feita clicando no componente de input e selecionando o arquivo através da janela nativa ou utilizando o drag and drop (arrastar e soltar).

O componente indica ao usuário o progresso do upload e fornece feedbacks sobre o status do envio.

## Boas práticas

- Sempre mantenha o nome do arquivo visível.
- Forneça informações sobre tamanho e tipo do arquivo, bem como velocidade de upload/download ou progresso.
