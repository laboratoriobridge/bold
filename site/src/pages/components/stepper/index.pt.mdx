# Stepper

O componente é uma indicação visual do progresso do usuário através de um conjunto de passos. Serve para guiar o usuário através de um conjunto de etapas para completar um processo específico, geralmente um Wizzard.

## Boas práticas

- Apresente os itens sequenciais da esquerda para a direita.
- Permita que o usuário volte em algum passo caso necessite revisar uma etapa.
- Sempre que possível utilize alguma indicação do status dos passos do wizard, como quando está completo/validado, se está incompleto/inválido.

## Situação dos Steps
