# Formulário

Formulários são usados para submeter dados. Todo campo deve vir acompanhado de uma descrição.

## Boas práticas

- É recomendável que a validação dos dados ocorra antes do usuário submeter o formulário, sempre que possível.
- A validação deve ocorrer assim que o usuário tirar o foco do campo ativo.
- Utilize placeholders para auxiliar na entrada de campos numéricos. Os placeholders ficam visíveis até que o usuário inicie a digitação.
- Utilize máscaras que ajudem o usuário a identificar o padrão do input.
- Inputs obrigatórios devem estar sinalizados com um asterisco vermelho após a descrição.

## Exemplo

O exemplo a seguir não usa nenhuma biblioteca para gerenciar o estado do formulário. Na prática, você pode (mas não precisa) usar uma biblioteca de terceiros para gerenciar o estado do formulário, como [`react-use-form-state`](https://github.com/wsmd/react-use-form- state), [`react-final-form`](https://github.com/final-form/react-final-form), [`formik`](https://github.com/jaredpalmer/formik) etc.
