# Multi Select

São usados quando há a opção de selecionar mais de um item de uma lista e conta com 5 ou mais itens. O componente permite que o usuário realize buscas na lista e também apresenta os itens que já foram selecionados dentro do campo, e possibilidade selecionar todos os itens de uma vez.

Se o número de caixas de comprimidos exceder a área do componente, use um [Tag](/components/tag) para indicar o número de itens ocultos selecionados (por exemplo: [+3]).

## Boas práticas

- A ordem dos itens da lista deve ser baseada na frequência de uso e, se aplicável, a em ordem crescente.
- Se a lista possuir menos de 5 itens, é recomendável que as opções sejam apresentadas diretamente na tela utilizando o componente de [Checkbox](/components/checkbox).
- Sempre que possível agrupe as opções e utilize títulos curtos que descrevam o conteúdo da categoria (Ex.: Em um combo de alimentos, as opções podem ser agrupadas por: vegetais, frutas, etc.).
- Para situações onde é possível selecionar apenas um item utilize [Select](/components/select) ou [Radio Button](/components/radio-button).
