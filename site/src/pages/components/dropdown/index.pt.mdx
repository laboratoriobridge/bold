# Dropdown

Utilizado para revelar uma lista de opções ou comandos ocultos. No cabeçalho, é geralmente utilizado para navegação, onde uma ação leva o usuário para outra página.

## Boas práticas

- Não remova itens da lista se essa ação não estiver disponível. Neste caso, mantenha o item desabilitado.
- Não coloque muitos itens na lista. Caso precise de muitas opções, considere utilizar outro tipo de cabeçalho/menu.
