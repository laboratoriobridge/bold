# Switch

É utilizado para trocar rapidamente entre dois possíveis estados. São muito usados para situações como "Ligado/Desligado" e ações que ocorrem imediatamente após o usuário mudar o _switch_.

## Boas práticas

- Utilize descrições junto ao componente para que a ação fique clara.
- Use adjetivos ao invés de verbos para descrever as labels e o estado do switch (Ex: Ligado, Desligado).
- A descrição deve permanecer no lado direito do switch.
- Sua função é habilitar ou desabilitar uma única função. Caso precise de um componente para fazer uma seleção entre mais opções, utilize [Checkbox](/components/checkbox) ou [Radio Button](/components/radio-button).
- No estado de _hover_, utilize _hint_ com verbos indicando ao usuário o resultado do status caso complete a ação (Ex.: Desabilitar envio automático).
