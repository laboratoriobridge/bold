# Table

O componente apresenta dados distribuídos em linhas e colunas, apresentando-os de uma forma que permite uma fácil leitura, comparação e escaneamento das informações.

## Boas práticas

- Colunas onde é possível fazer a ordenação devem apresentar o ícone de ordenação ao lado direito da label.
- O [Paginator](/components/paginator) só deve ser apresentado quando há paginação (mais de 10 itens).
