Paginator

Used to navigate between multiples pages whenever there are too many results to show at once or when something will take a long time to load/render all items. Is normally used in a table or to display search results.

Example

of 20
import { Paginator } from 'bold-ui'
import React from 'react'

function PaginatorEx() {
  return <Paginator page={4} total={20} onChange={console.log} />
}

export default PaginatorEx
